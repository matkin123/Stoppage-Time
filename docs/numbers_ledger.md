## O3 in-stoppage time-wasting gross-up (ADR-0024; central=ON, h=4, silent_marked)
- grosses up omitted CLOCK for the stoppage WITHIN added time, then applies the decayed lambda to the live portion; central is gross-up ON (one pass). Only the genuine-stoppage fraction z of dead time recurs (refs compensate stoppage, not normal flow), so one pass adds z*(1-live_share) of the clock and the geometric limit is ls/(1-z*(1-ls)) -- just above ON, NOT the old 1/live_share (ADR-0024 z-correction; z=0.38 from regulation dead vs counted stoppage).
- 1H+2H: gross-up off 21.4% -> on(CENTRAL) 24.8% -> geometric ceiling 26.0%
- 2H_only: gross-up off 14.5% -> on(CENTRAL) 17.0% -> geometric ceiling 18.0%

State census: tied 98 (7.86pp, p_trail-immune) + lead_by_1 121 (5.12pp, sensitive) + lead_by_2plus 95 (0) -> only 39.4% of the flip is p_trail-sensitive; flip/scoreline=0.524 is a state-mix identity.
